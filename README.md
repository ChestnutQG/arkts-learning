# ArkTS 学习项目

HarmonyOS 声明式 UI (ArkTS) 学习期间的练习代码。基于 DevEco Studio + API 12。

## 涵盖内容

- 组件化开发 (`@Component` / `@Entry`)
- 页面导航与路由
- 数据持久化 (Preferences / RDB)
- 图片缓存管理
- 瀑布流 / 网格布局
- 购物车状态管理

## 项目结构

```
├── business/          # 业务页面 (商品分类/购物车/店铺)
├── entity/            # 数据模型与工具类 (DataSource/Cache)
└── pages/             # 入口页面
```

## 运行

DevEco Studio 打开根目录 → Build → Run
